# Concept produit — ANIF Seller

[← Retour](README.md)

## Résumé

ANIF Seller est une application web qui aide les petits vendeurs à organiser leurs ventes réalisées sur WhatsApp, Instagram, Facebook et en boutique physique.

Le vendeur crée son catalogue, partage un lien de boutique à ses clients, reçoit les commandes dans un tableau de bord, suit les paiements et prépare ses livraisons.

## Problème

Beaucoup de petits commerces vendent déjà en ligne, mais sans outil structuré :

- les produits sont envoyés un par un dans des statuts WhatsApp ;
- les prix, tailles, couleurs et disponibilités sont répétés à chaque client ;
- les commandes sont perdues dans les conversations ;
- le stock est suivi mentalement ou dans un carnet ;
- les clients doivent demander plusieurs fois si un article est disponible ;
- les vendeurs oublient parfois de relancer les clients intéressés ;
- les paiements et livraisons sont suivis manuellement.

Ce fonctionnement peut marcher au début, mais il devient vite fatigant dès que le vendeur reçoit plusieurs commandes par jour.

## Solution

ANIF Seller donne au vendeur un espace simple pour :

- créer un catalogue de produits avec photos, prix, variantes et stock ;
- générer une page boutique publique ;
- partager le lien boutique sur WhatsApp, Instagram ou Facebook ;
- recevoir les commandes sans échange interminable ;
- suivre les statuts de commande ;
- générer des messages WhatsApp propres pour confirmer, relancer ou informer le client ;
- consulter les clients et commandes précédentes.

## Ce que l'application ne doit pas être au début

ANIF Seller ne doit pas démarrer comme une grosse marketplace compliquée. Le MVP doit éviter :

- marketplace multi-vendeurs publique ;
- gestion comptable complète ;
- application mobile native ;
- automatisation WhatsApp avancée avec API officielle ;
- système de livraison complexe ;
- CRM trop détaillé.

La première version doit rester légère : un vendeur, un catalogue, des commandes, un suivi clair.

## Différenciation

La force du produit vient de son adaptation au terrain :

- pensée pour WhatsApp, pas contre WhatsApp ;
- interface simple pour des vendeurs non techniques ;
- prix bas, en FCFA ;
- paiements locaux possibles avec FedaPay ou KkiaPay ;
- livraison locale configurable ;
- possibilité de réutiliser l'expérience ANIFOWOCHE e-commerce.

## Phrase de vente

> Crée ta boutique en ligne, partage le lien sur WhatsApp, reçois tes commandes dans un tableau clair.

## Hypothèse principale à valider

Les petits vendeurs sont prêts à payer un abonnement mensuel modéré si l'application leur fait gagner du temps, évite les oublis de commandes et donne une image plus professionnelle à leur activité.
